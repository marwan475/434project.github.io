About

IP2ORG2IP is an IP to organization name and organization name to ip resolution service. Our goal is to research how IP to Organization attribution services and DNS resolution works by implementing our own service that provides both, while trying to fix some of the issues that they currently face.


Features

IP2ORG lookup where you can input a set of IP's and dates, then retrieve the corresponding organization names. ORG2IP lookup where you can input a set of organization names and dates then get the corresponding IP's. These together provide historic, bulk attribution services, which addresses the main issue of the current day attribution services.


Importance

IP2ORG and ORG2IP resolution services play a crucial role in the operation of the Internet, providing valuable information regarding domain names, IP addresses, organization information, and autonomous systems. These services are particularly useful in the field of networking, being used in network troubleshooting and investigation of cyber security incidents.
